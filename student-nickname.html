จากโค้ดฝั่ง Google Apps Script และหน้า HTML ที่คุณให้มา ระบบทำงานได้ตามความต้องการหลักของคุณแล้ว โดยสรุปการทำงานและข้อเสนอแนะมีดังนี้:

✅ การทำงานปัจจุบัน:

  1. ค้นหานักเรียนด้วยเลขบัตรประชาชน:

    • เมื่อพิมพ์เลขบัตรประชาชนแล้วกด “ค้นหา” (searchStudent()), จะส่งคำขอ GET ไปยัง Google Apps Script.

    • ฝั่ง Apps Script ตรวจสอบข้อมูลในชีท GenID และส่งข้อมูลกลับมา (ชื่อ, นามสกุล, รหัสนักเรียน, ชื่อเล่น).

  2. แสดงข้อมูลในฟอร์ม:

    • หากพบข้อมูล, จะแสดงในฟอร์มพร้อมให้แก้ไขชื่อเล่นได้.

    • หากไม่พบ, จะแสดงข้อความ “ไม่พบนักเรียนในระบบ”.

  3. บันทึกชื่อเล่น:

    • เมื่อกรอกชื่อเล่นและกด “บันทึก”, จะส่งคำขอ POST ไปยัง Google Apps Script.

    • ฝั่ง Apps Script ควรอัปเดตชื่อเล่นในชีท GenID ตามเลขบัตรประชาชน.

⚠️ จุดที่ควรแก้ไขหรือปรับปรุง:

1. saveNickname() ไม่ทำงานเพราะโค้ดใน <script> ซ้อนกันผิด

ใน <script> ฟังก์ชัน saveNickname() มีโครงสร้างไม่ถูกต้อง มี try { try { ซ้อนกัน และฟังก์ชัน saveFormData() ถูกนิยามซ้ำโดยไม่เรียกใช้

✅ ควรปรับให้เรียบง่ายแบบนี้:

javascript
async function saveNickname() {
  const idCard = document.getElementById('idCard').value.trim();
  const nickname = document.getElementById('nickname').value.trim();

  if (!nickname) {
    alert("กรุณากรอกชื่อเล่นก่อนบันทึก");
    return;
  }

  try {
    const urlEncodedData = new URLSearchParams();
    urlEncodedData.append("idCard", idCard);
    urlEncodedData.append("nickname", nickname);
    urlEncodedData.append("action", "updateNickname");

    await fetch(API_URL, {
      method: 'GET', // ต้องใช้ GET ตาม Apps Script ที่รองรับไว้
      headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
    });

    alert("✅ บันทึกชื่อเล่นเรียบร้อย");
  } catch (error) {
    console.error("เกิดข้อผิดพลาด", error);
    alert("❌ ไม่สามารถบันทึกข้อมูลได้");
  }
}

2. doPost() ไม่รองรับ action=updateNickname ตามที่ฝั่ง HTML ต้องการ

แต่ใน doGet() รองรับ action=updateNickname แล้ว ดังนั้นการบันทึกชื่อเล่นควรใช้ GET (แม้จะดูผิดหลัก REST แต่ก็ใช้ได้ใน Apps Script)

✅ สรุปแนะนำการใช้งาน:

  • ใช้ เลขบัตรประชาชน ค้นหานักเรียน

  • ถ้าพบชื่อ, นามสกุล, รหัสนักเรียน → จะแสดงในฟอร์ม

  • ถ้าไม่มีชื่อเล่น → พิมพ์ชื่อเล่นและกด “บันทึก”

  • ระบบจะอัปเดตกลับเข้า Google Sheet โดยใช้ GET + action=updateNickname

หากต้องการให้ช่วยปรับโค้ด HTML หรือ Apps Script ให้เรียบร้อยและปลอดภัยยิ่งขึ้น แจ้งได้เลยครับ!
